# Airplane Display Pi

This project displays real-time airplane information on a 64x64 RGB LED matrix using a Raspberry Pi.

## Features

- Fetches flight data from the OpenSky Network API.
- Displays airplane callsigns, altitudes, and other relevant information.

## Hardware

- Raspberry Pi Zero 2 W
- 64x64 RGB Full-Colour LED Matrix Panel
- Adafruit RGB Matrix Bonnet
